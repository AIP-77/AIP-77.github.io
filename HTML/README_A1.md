# Аналитика по видам работ (A1.html)

Веб-приложение для анализа и визуализации данных о выполненных работах сотрудников склада/логистического центра.

## 📋 Описание

Страница `A1.html` предоставляет интерактивную аналитику по видам работ с возможностью:
- Просмотра данных за конкретные даты через календарь
- Анализа общей статистики за день
- Детальных графиков и диаграмм
- Группировки по отделам и видам работ
- Экспорта данных в Excel

**Особенность:** При первой загрузке автоматически отображаются данные за **последнюю доступную дату** из архива.

- `formatTime(seconds)` — форматирование секунд в "HH:MM"
- `formatDate(date)` — форматирование даты в "DD.MM.YYYY"

- `isResponsible(position)` — проверка, является ли должность ответственной

### 2. `data.js` — Загрузка и обработка данных
**Назначение:** Работа с внешними источниками данных (архивы, нормативы, персонал)

**Основные функции:**

- `loadStaffData()` — загрузка данных персонала из `staff.json`

- `getLastAvailableDate()` — **получение последней доступной даты** из архива

- `getArchiveUrl(archiveName)` — формирование URL для загрузки архива

## 🚀 Как работает автозагрузка последней даты

### Алгоритм (`data.js` → `getLastAvailableDate()`):

1. Определяется текущий архив (YYYY-MM)
2. Загружаются данные из архива
3. Извлекаются все уникальные даты из поля `'Рабочий день'`
4. Даты сортируются по убыванию (от новых к старым)
5. Возвращается первая дата (самая свежая)

### Применение (`A_script.js` → `initUI()`):

```javascript
// Если selectedDate ещё не установлена, используем последнюю доступную дату
const uniqueDates = [...new Set(records.map(r => r['Рабочий день']))].filter(Boolean).sort();
if (uniqueDates.length > 0 && !selectedDate) {
  // Сортируем даты по убыванию и берём первую (последнюю по времени)
  uniqueDates.sort((a, b) => {
    const [dayA, monthA, yearA] = a.split('.').map(Number);
    const [dayB, monthB, yearB] = b.split('.').map(Number);
    const dateA = new Date(yearA, monthA - 1, dayA);
    const dateB = new Date(yearB, monthB - 1, dayB);
    return dateB - dateA;
  });
  selectedDate = uniqueDates[0];
  renderReport();
}
```

## 🔧 Установка и запуск

### Требования:
- Веб-сервер (для работы с CORS и fetch API)
- Доступ к папке `/archive/` с данными

### Локальный запуск:

```bash
# Вариант 1: Python
cd /workspace/HTML
python -m http.server 8000

# Вариант 2: Node.js (требуется http-server)
npx http-server ./HTML -p 8000

# Вариант 3: PHP
php -S localhost:8000 -t ./HTML
```

Откройте в браузере: `http://localhost:8000/A1.html`

### Развёртывание на GitHub Pages:
